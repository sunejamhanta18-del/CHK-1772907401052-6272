AgriPulse: A Smart Monitoring Solution for Sustainable Onion Storage

AgriPulse is a mobile application developed to help farmers minimize post-harvest losses.By monitoring environmental conditions in real-time, the app predicts and identifies onion rot before it spreads, ensuring better crop quality and higher profits for farmers.

Key Features:
Real-time Climate Monitoring: Tracks temperature and humidity to maintain optimal storage conditions.
Dynamic Stats & Refresh: Simulates live sensor data with automatic updates every 30 minutes (demoed at 5-second intervals).
Rate of Change Visualization: Includes interactive line charts to track sensor trends over time using MPAndroidChart.
Smart Shelf Management: A 6-shelf grid system with color-coded alerts (Green/Yellow/Red) that provide specific causes of rot (e.g., Bacterial Soft Rot) upon selection.
Multilingual Interface: Fully localized in English, Hindi (हिन्दी), and Marathi (मराठी) to ensure accessibility for regional farmers.

Technical Implementation:
Platform: Android (Min SDK 23, Target SDK 36)

Language: Java

Architecture: Activity-based navigation with a custom Navigation Drawer.

Libraries:

MPAndroidChart for data visualization.

Material Components for modern UI/UX.

Firebase (Integrated for future cloud sync).
